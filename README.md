# Football Player Bounding Box Annotation & Tracking Using CVAT

## Project Overview

This portfolio project demonstrates a sports video annotation workflow completed using CVAT. The task involved annotating visible on-field football players from both teams using accurate bounding boxes and consistent player tracking IDs.

The dataset included 599 frames, where each visible player needed tight bounding boxes, team-based labeling, and stable identity tracking across frames.

## Annotation Workflow

* Reviewed the full video sequence to understand player movement and scene changes
* Identified visible on-field players from both teams
* Annotated players using tight bounding boxes
* Assigned team-based labels and consistent tracking IDs
* Adjusted boxes during movement, overlaps, and partial visibility
* Reviewed the full sequence before final delivery
